# Insurence-Analytics

## Overview
This project analyzes insurance business performance using sales and policy data.
The goal is to track KPIs such as New Business, Renewals, Cross-Sell, and Conversion Ratios.

## Tools & Technologies
- SQL (MySQL)
- Excel
- Power BI
- Tableau

## Key KPIs
- New, Renewal & Cross-Sell Performance
- Invoice vs Target Achievement
- Conversion Ratio
- Branch & Account Executive Analysis

## Outcome
Created interactive dashboards and SQL-based KPI reports to support
data-driven decision-making for insurance business teams.
